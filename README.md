### Project 4: Restaurant Website
**Description:**  
A responsive restaurant landing page built using HTML and CSS. The website features a sticky navigation bar, a hero section with promotional content, a features section, a visually engaging menu layout using CSS Grid, and a structured footer. This project helped me strengthen my understanding of layout techniques, responsive design, and modern CSS styling.

**HTML Concepts Used:**  
- Semantic HTML structure (`nav`, `section`, `footer`)
- Headings and text content  
- Lists for navigation and footer links
- Buttons and form input (`button`, `input[type="email"]`) 
- Icons using Font Awesome

**CSS Concepts Used:**  
- Parallax scrolling using `background-attachment: fixed`  
- Flexbox for layout (navigation, hero section, features, footer)  
- CSS Grid with grid-template-areas for menu layout  
- Responsive design using media queries  
- Typography styling (Google Fonts, font size, weight, line height)  
- Hover effects and transitions
- Positioning (sticky navigation, z-index)
