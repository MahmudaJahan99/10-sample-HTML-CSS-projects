### Project 3: Parallax Website
**Description:**  
This project demonstrates the parallax effect using fixed background images while layering text and content sections on top. It focuses on visual presentation, section-based layout, and creating depth using CSS background properties.

**HTML Concepts Used:**  
- Semantic layout using `section` elements  
- Structured content separation  
- Text overlay elements  
- Headings and paragraphs for content organization

**CSS Concepts Used:**  
- Parallax scrolling using `background-attachment: fixed`  
- Background images and positioning  
- Absolute and relative positioning  
- Typography styling and letter spacing  
- Color contrast for light and dark sections  
- Reusable utility classes
